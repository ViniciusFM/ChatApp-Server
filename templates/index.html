ChatApp Server — API Backend for ChatApp

ChatApp Server is the backend service that powers the mobile application ChatApp. It provides a API that handles messaging, authentication, and chat channels functionalities for the app.

This server was developed as part of an educational project in the Mobile Device Programming course at IFTM - Campus Uberaba Parque Tecnológico (UPT), taught by Vinícius F. Maciel. Both the ChatApp mobile client and the ChatApp Server backend were designed to explore development practices in mobile and web technologies.

The server is released under the GNU Affero General Public License (AGPLv3), ensuring transparency, open collaboration, and access to the source code for further learning and improvement.
